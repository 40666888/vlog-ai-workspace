# 晨间提醒日历 · 长期保存版

这是一个纯静态网页版本的提醒日历，适合记录每天早上要做的事情，并自动生成待办事项、显示完成进度、查看周/月/年完成率。

它的设计重点不是“在线演示”，而是“适合长期保存和备份”：

- 不依赖外部 CDN、数据库或服务器
- `HTML / CSS / JS` 分离，便于长期维护
- 数据默认保存在浏览器本地
- 支持导出 JSON 备份，方便迁移和长期归档
- 整个文件夹可以直接复制、压缩、备份到网盘或移动硬盘

## 功能说明

- 记录晨间计划：把每天要做的事情按行写入，自动生成待办任务
- 单条补充录入：临时新增任务也可以快速加入
- 任务进度可调：每条任务都能直接选择 `0% / 25% / 50% / 75% / 100%`
- 未完成事项可视化：列表中会显示当前进度条
- 周 / 月 / 年进度表：自动统计已完成、进行中、未开始和完成率
- 月历视图：查看某一天安排了多少任务、完成了多少任务
- 提醒时间线：按时间顺序查看当天任务
- 浏览器提醒：授权后，在网页保持打开时会按提醒时间弹出通知
- 导出 / 导入备份：适合长期保存和恢复

## 使用方式

1. 直接双击打开 `index.html`，或者用浏览器打开它。
2. 在“晨间事项输入”里，一行写一条任务。
3. 点击“生成今日代办”。
4. 到任务清单里直接调整每条任务的完成进度。
5. 在周/月/年统计区查看整体完成率。

## 输入格式示例

支持以下几种常见写法：

```text
07:30 晨跑 30 分钟 | 健康
09:00 跟进客户报价 | 工作 | 先确认合同
整理报销资料 | 财务 | 16:30 | 核对发票和截图
```

说明：

- 开头写 `09:00` 会被识别为提醒时间
- `| 分类 |` 会被识别为任务分类
- 其他内容会被当作备注

## 长期保存与备份建议

推荐同时做两层保存：

1. 日常使用时，网页会自动保存到浏览器本地。
2. 每周或每次重要更新后，点击“导出备份”下载一个 JSON 文件。

建议把以下两部分一起归档：

- 整个 `reminder-calendar-v1/` 文件夹
- 导出的 JSON 备份文件

这样即使换浏览器、换电脑，也可以：

1. 把整个文件夹复制过去
2. 打开 `index.html`
3. 点击“导入备份”
4. 恢复之前的数据

## 版本保存建议

如果你后续还要继续升级这个网页，建议不要覆盖当前版本，而是按下面方式保留：

- `reminder-calendar-v1/`
- `reminder-calendar-v2/`
- `reminder-calendar-v3/`

这样更适合长期留档和回滚，也符合“不要覆盖旧页面”的要求。

## 提醒功能说明

浏览器提醒有一个限制：

- 只有网页保持打开时，浏览器通知才能按提醒时间弹出

如果你更看重“长期保存”和“稳定备份”，当前这种纯静态结构是比较稳妥的方案。
